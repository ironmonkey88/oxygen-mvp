How it works

Built on Oxygen

Somerville's 311 SODA feed is ingested via dlt to Parquet, transformed through bronze and gold dbt layers in DuckDB, and surfaced through the Airlayer semantic layer. The chat interface is an Oxygen Answer Agent backed by Claude Sonnet 4.6.

The agent operates under a trust contract: every reply includes the executed SQL, the row count returned, citations to source tables and Airlayer views, and surfaced entries from the limitations registry when the query touches a flagged area. Methodology is always inspectable.

Out of scope for MVP 1: charts, exports, dashboards, follow-up suggestions, and resident-facing framing. Those land in MVP 2 and beyond.
